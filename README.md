## What is Hooked?

### Contribute Hooked

Clone repo and setup new feature/XXX branch

Now choose dev environment, if virtualenvwrapper is installed:

      make setup2 NAME=hook # for python 2.X
     
      make setup3 NAME=hook # for python 3.X
      
Now init

      make init
      

Test

      make test
