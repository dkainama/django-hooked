## What is Hooked?

### Contribute Hooked

Clone repo and branch of master

Now choose dev environment, if virtualenvwrapper is installed:

      make setup2 NAME=hook # for python 2.7
     
      make setup3 NAME=hook # for python 3.6
      
Now init

      make init
      

Test

      make test
